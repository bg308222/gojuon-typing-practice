# 五十音打字練習 - 功能規格文件

## 核心概念

一個讓使用者練習日文五十音羅馬字輸入的練習工具。使用者可以自由選擇要練習的假名範圍，系統僅在選取範圍內出題。

---

## 1. 假名選擇（Kana Selection）

### 1.1 可選範圍
- 支援平假名（ひらがな）與片假名（カタカナ），可分開或同時選擇
- 涵蓋全部五十音行：あ行、か行、さ行、た行、な行、は行、ま行、や行、ら行、わ行、ん
- 濁音（が行、ざ行、だ行、ば行）與半濁音（ぱ行）可獨立勾選
- 拗音（きゃ、しゃ、ちゃ … 等複合音）可獨立勾選

### 1.2 選擇方式
- 可以以「行」為單位勾選（例如：整個か行）
- 也可以以「單個假名」為單位勾選
- 提供快速選擇按鈕：全選、清除全部、只選平假名、只選片假名

### 1.3 限制
- 至少須選取 1 個假名才可開始練習
- 所選假名清單持久化於 IndexedDB，下次開啟自動沿用

---

## 2. 練習模式（Practice Session）

### 2.1 出題邏輯
- 每題從使用者選取的假名集合中隨機抽取一個
- 預設採「加權隨機」：答錯次數越多、正確率越低的假名出現機率越高，強化弱點練習
- 可選擇切換為「純隨機」（均等機率）

### 2.2 輸入方式
- 使用者以鍵盤輸入任何能代表該假名的字串，視為正確答案，包含：
  - **羅馬字（寬鬆匹配）**：接受所有合法拼法，例如：
    - し → `si` 或 `shi`
    - つ → `tu` 或 `tsu`
    - ち → `ti` 或 `chi`
    - ふ → `fu` 或 `hu`
  - **直接輸入假名字元**：輸入對應的平假名（如 `し`）或片假名（如 `シ`）皆視為正確；若題目為平假名，輸入對應的片假名也算對，反之亦然
- 判斷規則：只要輸入字串屬於該音節的合法表示之一，即判定答對
- 輸入完成後按 Enter 確認，或在輸入完整正確答案後自動跳下一題（可設定）

### 2.3 即時回饋
- 輸入過程中逐字元顯示正確 / 錯誤（類似打字練習的即時 highlight）
- 答對：立即進入下一題
- 答錯：顯示正確答案，記錄該假名本次答錯，按任意鍵繼續

### 2.4 練習結束條件
- 模式 A — 題數模式：預先設定題數（預設 20 題），答完後顯示結果
- 模式 B — 時間模式：預先設定時間（預設 60 秒），倒數結束後顯示結果
- 模式 C — 無限模式：不設終止條件，手動結束
- 三種模式可在開始前選擇

---

## 3. 統計與進度追蹤（Statistics）

### 3.1 單次練習結果
- 總題數、答對題數、答錯題數
- 正確率（%）
- 每題平均作答時間（ms）
- 最快 / 最慢作答的假名
- 答錯清單：列出錯誤的假名及使用者輸入的錯誤答案

### 3.2 歷史記錄
- 記錄每次練習的日期、模式、選取範圍、正確率、平均速度
- 可查看過去 N 次練習的趨勢（正確率折線、速度折線）

### 3.3 假名別統計
- 每個假名的累計出現次數、答對次數、正確率、平均反應時間
- 可依正確率排序，快速找出最弱的假名

### 3.4 資料儲存
- **所有資料統一使用 IndexedDB 儲存**，包含使用者設定、假名別統計、練習歷史
- 提供匯出（JSON）與清除功能

---

## 4. 設定（Settings）

| 設定項目 | 說明 | 預設值 |
|---|---|---|
| 出題模式 | 加權隨機 / 純隨機 | 加權隨機 |
| 自動進題 | 輸入正確後是否自動跳題（不需按 Enter） | 開啟 |
| 顯示假名提示 | 題目是否同時顯示假名的發音提示（羅馬字小字） | 關閉 |
| 練習模式 | 題數 / 時間 / 無限 | 題數 |
| 預設題數 | 題數模式的預設題目數 | 20 |
| 預設時間 | 時間模式的預設秒數 | 60 |
| 輸入法容錯 | 是否接受多種合法羅馬字拼法 | 開啟 |

---

## 5. 鍵盤操作

- 練習中全程可用鍵盤操作，不需要滑鼠
- 開始練習：Enter
- 答錯後繼續下一題：Enter 或 Space
- 練習中途暫停 / 繼續：Esc
- 從結果頁回到設定：Backspace 或 R 鍵

## 7. 邊界情境處理

- **ん（n）的判斷**：當使用者輸入單一 `n` 後，若下一個字元不是母音或 `y`，才確認為「ん」；否則等待下一個字元。接受 `nn` 作為強制確認輸入。
- **大小寫不敏感**：輸入 `Ka`、`KA`、`kA` 皆視為有效。
- **假名切換一致性**：同一個練習 session 中，若同時選取平假名與片假名，同一個音節的平片假名視為同一題（例如か與カ為同一音，不重複出題）；若想分開練習，則在選擇時分別只選平假名或片假名。
